## Создание новой книги

Начнем с создания новой книги Excel с использованием Aspose.Cells для Java. Вот базовый фрагмент кода, с которого можно начать:

```java
// Инициализировать новую книгу
Workbook workbook = new Workbook();

// Добавить рабочий лист
Worksheet worksheet = workbook.getWorksheets().get(0);

// Сохраните книгу
workbook.save("sample.xlsx");
```

Этот код создает новую книгу и сохраняет ее как «sample.xlsx».

## Добавление данных на рабочий лист

Теперь, когда у нас есть рабочая книга, нам нужно добавить в нее некоторые данные. Вот как можно добавлять числа в ячейки на листе:

```java
// Доступ к ячейке и добавление данных
Cell cell = worksheet.getCells().get("A1");
cell.putValue(10);

// Сохраните книгу
workbook.save("sample.xlsx");
```

В этом примере мы добавили число 10 в ячейку A1.

## Понимание формулы СУММ

 Формула СУММ используется для расчета суммы диапазона чисел в Excel. Его основной синтаксис`=SUM(range)`, где «диапазон» представляет ячейки, которые вы хотите сложить.

## Использование функциональности SUM с Aspose.Cells

Aspose.Cells упрощает реализацию формулы СУММ. Вот как вы можете его использовать:

```java
// Суммировать значения в диапазоне
Cell sumCell = worksheet.getCells().get("B1");
sumCell.setFormula("=SUM(A1:A10)");

// Рассчитаем и сохраним книгу.
workbook.calculateFormula();
workbook.save("sample.xlsx");
```

 В этом примере мы использовали`setFormula`метод, позволяющий применить формулу СУММ к ячейке B1, суммируя значения в ячейках от A1 до A10.

## Применение суммы в разных диапазонах

Вы также можете применить формулу СУММ к нескольким диапазонам на листе. Например, если у вас есть данные в разных столбцах или строках, которые вы хотите добавить отдельно, вы можете сделать это следующим образом:

```java
// Суммируйте два разных диапазона
Cell sumCell1 = worksheet.getCells().get("B1");
sumCell1.setFormula("=SUM(A1:A10)");

Cell sumCell2 = worksheet.getCells().get("C1");
sumCell2.setFormula("=SUM(D1:D10)");

// Рассчитаем и сохраним книгу.
workbook.calculateFormula();
workbook.save("sample.xlsx");
```

Здесь мы вычислили сумму значений в ячейках от A1 до A10 и от D1 до D10 и поместили результаты в ячейки B1 и C1 соответственно.

## Условная сумма с Aspose.Cells

 Aspose.Cells также позволяет реализовывать формулы условной суммы, которые могут быть чрезвычайно полезны для сложного анализа данных. Вы можете использовать такие функции, как`SUMIF` и`SUMIFS` применить условия к вашим суммам.

```java
// Условная СУММА
Cell sumCell = worksheet.getCells().get("B1");
sumCell.setFormula("=SUMIF(A1:A10, \">5\")");

// Рассчитаем и сохраним книгу.
workbook.calculateFormula();
workbook.save("sample.xlsx");
```

В этом примере мы суммируем значения в ячейках от A1 до A10, но включая только числа больше 5.
