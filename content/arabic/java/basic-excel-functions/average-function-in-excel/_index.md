## مقدمة إلى الدالة المتوسطة في Excel

تُستخدم جداول بيانات Excel على نطاق واسع لتحليل البيانات والحسابات. إحدى الوظائف الأكثر استخدامًا للتحليل الرقمي هي الدالة AVERAGE، والتي تتيح لك العثور على متوسط نطاق من الأرقام. في هذه المقالة، سنستكشف كيفية استخدام الدالة AVERAGE في Excel باستخدام Aspose.Cells for Java، وهي واجهة برمجة تطبيقات قوية للعمل مع ملفات Excel برمجيًا.

## إعداد Aspose.Cells لجافا

قبل أن نتعمق في استخدام الدالة AVERAGE، نحتاج إلى إعداد بيئة التطوير الخاصة بنا. اتبع هذه الخطوات للبدء:

1.  تنزيل Aspose.Cells لـ Java: تفضل بزيارة[Aspose.Cells لجافا](https://releases.aspose.com/cells/java/) لتحميل المكتبة.

2.  تثبيت Aspose.Cells: اتبع تعليمات التثبيت المتوفرة في وثائق Aspose[هنا](https://reference.aspose.com/cells/java/).

بمجرد تثبيت Aspose.Cells for Java، تصبح جاهزًا لبدء العمل مع ملفات Excel.

## إنشاء مصنف Excel جديد

لاستخدام الدالة AVERAGE، نحتاج أولاً إلى مصنف Excel. لنقم بإنشاء واحدة برمجيًا باستخدام Aspose.Cells:

```java
// كود Java لإنشاء مصنف Excel جديد
Workbook workbook = new Workbook();
Worksheet worksheet = workbook.getWorksheets().get(0);
```

في هذا الكود، نقوم بإنشاء مصنف جديد والوصول إلى ورقة العمل الأولى.

## إضافة البيانات إلى المصنف

والآن بعد أن أصبح لدينا مصنف، فلنضيف بعض البيانات إليه. سنقوم بمحاكاة مجموعة بيانات من الأرقام:

```java
// كود جافا لإضافة البيانات إلى مصنف Excel
worksheet.getCells().get("A1").putValue(10);
worksheet.getCells().get("A2").putValue(20);
worksheet.getCells().get("A3").putValue(30);
worksheet.getCells().get("A4").putValue(40);
```

هنا، نقوم بملء الخلايا من A1 إلى A4 بقيم عددية.

## استخدام الدالة المتوسطة

تقوم الدالة AVERAGE في Excel بحساب متوسط نطاق من الأرقام. باستخدام Aspose.Cells for Java، يمكنك تحقيق ذلك بسهولة برمجيًا:

```java
// كود جافا لحساب المتوسط باستخدام Aspose.Cells
Cell cell = worksheet.getCells().get("B1");
cell.setFormula("=AVERAGE(A1:A4)");
```

في هذا الكود، قمنا بتعيين صيغة الخلية B1 لحساب متوسط الأرقام في الخلايا من A1 إلى A4.

## تنسيق ورقة إكسل

يمكنك تنسيق ورقة Excel حسب متطلباتك. قم بتغيير الخطوط والألوان والأنماط بسهولة باستخدام Aspose.Cells. على سبيل المثال:

```java
// كود جافا لتنسيق ورقة Excel
Style style = cell.getStyle();
style.getFont().setName("Arial");
style.getFont().setSize(12);
style.setForegroundColor(Color.getRed());
cell.setStyle(style);
```

يقوم هذا الكود بتغيير الخط والحجم واللون الأمامي للخلية.

## حفظ وتصدير ملفات Excel

بمجرد إنشاء ورقة Excel وتنسيقها، يمكنك حفظها في موقع محدد أو تصديرها إلى تنسيقات مختلفة مثل PDF أو CSV. وإليك كيفية حفظه بصيغة PDF:

```java
// كود جافا لحفظ المصنف بصيغة PDF
workbook.save("output.pdf", SaveFormat.PDF);
```

يحفظ هذا الرمز المصنف كملف PDF.
