غالبًا ما تُستخدم حماية جدول بيانات Excel لتقييد الوصول إلى البيانات وتعديلها. في هذا البرنامج التعليمي ، سنوجهك خطوة بخطوة لفهم وتنفيذ كود المصدر C # المقدم لإلغاء تأمين جدول بيانات Excel محمي باستخدام مكتبة Aspose.Cells لـ .NET.

## الخطوة الأولى: تهيئة البيئة

قبل أن تبدأ ، تأكد من تثبيت Aspose.Cells for .NET على جهازك. يمكنك تنزيل المكتبة من الموقع الرسمي لشركة Aspose وتثبيتها باتباع التعليمات المتوفرة.

بمجرد اكتمال التثبيت ، قم بإنشاء مشروع C # جديد في بيئة التطوير المتكاملة المفضلة لديك (IDE) واستورد مكتبة Aspose.Cells لـ .NET.

## الخطوة 2: تكوين مسار دليل المستند

 في التعليمات البرمجية المصدر المتوفرة ، تحتاج إلى تحديد مسار الدليل حيث يوجد ملف Excel الذي تريد فتحه. تعديل`dataDir` متغير عن طريق استبدال "دليل المستند" بالمسار المطلق للدليل على جهازك.

```csharp
// المسار إلى دليل المستندات.
string dataDir = "PATH TO YOUR DOCUMENTS DIRECTORY";
```

## الخطوة 3: إنشاء كائن مصنف

للبدء ، نحتاج إلى إنشاء كائن مصنف يمثل ملف Excel الخاص بنا. استخدم مُنشئ فئة المصنف وحدد المسار الكامل لملف Excel لفتحه.

```csharp
// إنشاء كائن مصنف
Workbook workbook = new Workbook(dataDir + "book1.xls");
```

## الخطوة 4: الوصول إلى جدول البيانات

 بعد ذلك ، نحتاج إلى الانتقال إلى ورقة العمل الأولى في ملف Excel. استخدم ال`Worksheets` الخاص بكائن المصنف للوصول إلى مجموعة أوراق العمل ، ثم استخدم ملحق`[0]` فهرس للوصول إلى الورقة الأولى.

```csharp
// الوصول إلى ورقة العمل الأولى في ملف Excel
Worksheet worksheet = workbook.Worksheets[0];
```

## الخطوة 5: فتح جدول البيانات

 الآن سنقوم بإلغاء تأمين ورقة العمل باستخدام ملف`Unprotect()` طريقة كائن ورقة العمل. اترك سلسلة كلمة المرور فارغة (`""`) إذا لم يكن جدول البيانات محميًا بكلمة مرور.

```csharp
// إلغاء حماية ورقة العمل بكلمة مرور
worksheet.Unprotect("");
```

## الخطوة 6: حفظ ملف Excel غير المؤمّن

بمجرد إلغاء قفل جدول البيانات ، يمكننا حفظ ملف Excel النهائي. استخدم ال`Save()` طريقة لتحديد المسار الكامل لملف الإخراج.

```csharp
// حفظ المصنف


workbook.Save(dataDir + "output.out.xls");
```

### نموذج التعليمات البرمجية المصدر لإلغاء تأمين ورقة Excel المحمية باستخدام Aspose.Cells لـ .NET 
```csharp
try
{
    // المسار إلى دليل المستندات.
    string dataDir = "YOUR DOCUMENT DIRECTORY";
    // إنشاء كائن مصنف
    Workbook workbook = new Workbook(dataDir + "book1.xls");
    // الوصول إلى ورقة العمل الأولى في ملف Excel
    Worksheet worksheet = workbook.Worksheets[0];
    // إلغاء حماية ورقة العمل بكلمة مرور
    worksheet.Unprotect("");
    // حفظ المصنف
    workbook.Save(dataDir + "output.out.xls");
}
catch(Exception ex)
{
    Console.WriteLine(ex.Message);
    Console.ReadLine();
}
```
