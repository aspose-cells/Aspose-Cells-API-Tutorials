## Βήμα 1: Δημιουργήστε μια παρουσία της κλάσης Βιβλίο εργασίας

Για να ξεκινήσουμε, πρέπει να δημιουργήσουμε μια παρουσία της κλάσης Βιβλίο εργασίας, η οποία αντιπροσωπεύει το βιβλίο εργασίας του Excel. Αυτό μπορεί να επιτευχθεί χρησιμοποιώντας τον ακόλουθο κώδικα:

```csharp
Workbook book = new Workbook();
```

## Βήμα 2: Πρόσβαση στο υπολογιστικό φύλλο

Στη συνέχεια, πρέπει να πλοηγηθούμε στο φύλλο εργασίας του βιβλίου εργασίας όπου θέλουμε να ορίσουμε τις διαστάσεις της σελίδας. Σε αυτό το παράδειγμα, ας υποθέσουμε ότι θέλουμε να δουλέψουμε με το πρώτο φύλλο εργασίας. Μπορούμε να έχουμε πρόσβαση χρησιμοποιώντας τον ακόλουθο κώδικα:

```csharp
Worksheet sheet = book.Worksheets[0];
```

## Βήμα 3: Ρυθμίστε το μέγεθος χαρτιού σε A2 και το πλάτος και το ύψος εκτύπωσης σε ίντσες

Τώρα θα ορίσουμε το μέγεθος χαρτιού σε A2 και θα εκτυπώσουμε το πλάτος και το ύψος της σελίδας σε ίντσες. Αυτό μπορεί να επιτευχθεί χρησιμοποιώντας τον ακόλουθο κώδικα:
