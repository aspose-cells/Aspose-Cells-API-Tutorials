## Βήμα 1: Ορίστε τον Κατάλογο Εγγράφων

Για να ξεκινήσετε, πρέπει να ορίσετε τη διαδρομή καταλόγου όπου βρίσκεται το αρχείο Excel. Αντικαταστήστε τον "ΚΑΤΑΛΟΓΟ ΕΓΓΡΑΦΩΝ ΣΑΣ" στον κώδικα με την πραγματική διαδρομή του αρχείου σας Excel.

```csharp
//Η διαδρομή προς τον κατάλογο εγγράφων.
string dataDir = "YOUR DOCUMENT DIRECTORY";
```

## Βήμα 2: Ορίστε τη διαδρομή εισαγωγής αρχείου Excel

Στη συνέχεια, πρέπει να ορίσετε τη διαδρομή εισόδου του αρχείου Excel που θέλετε να ανοίξετε. Αυτή η διαδρομή θα χρησιμοποιηθεί για τη δημιουργία μιας ροής αρχείων.

```csharp
// Διαδρομή εισαγωγής αρχείου Excel
string InputPath = dataDir + "book1.xlsx";
```

## Βήμα 3: Δημιουργήστε μια ροή αρχείων και ανοίξτε το αρχείο Excel

 Στη συνέχεια, πρέπει να δημιουργήσετε μια ροή αρχείων και να ανοίξετε το αρχείο Excel χρησιμοποιώντας το`FileStream` τάξη.

```csharp
// Δημιουργήστε μια ροή αρχείου που περιέχει το αρχείο Excel για άνοιγμα
FileStream fstream = new FileStream(InputPath, FileMode.Open);
```

## Βήμα 4: Δημιουργήστε ένα αντικείμενο βιβλίου εργασίας

 Μετά το άνοιγμα του αρχείου Excel, πρέπει να δημιουργήσετε instantiation a`Workbook`αντικείμενο. Αυτό το αντικείμενο αντιπροσωπεύει το βιβλίο εργασίας του Excel και προσφέρει διάφορες μεθόδους και ιδιότητες για τον χειρισμό του βιβλίου εργασίας.

```csharp
// Δημιουργήστε ένα αντικείμενο βιβλίου εργασίας
// Ανοίξτε το αρχείο Excel μέσω της ροής αρχείων
Workbook workbook = new Workbook(fstream);
```

## Βήμα 5: Πρόσβαση σε ένα φύλλο εργασίας με όνομα

Για να αποκτήσετε πρόσβαση σε ένα συγκεκριμένο φύλλο εργασίας με όνομα, μπορείτε να χρησιμοποιήσετε το`Worksheets` ιδιοκτησία του`Workbook` αντικείμενο και ευρετηρίαση του ονόματος του φύλλου εργασίας.

```csharp
// Πρόσβαση σε ένα φύλλο εργασίας χρησιμοποιώντας το όνομα του φύλλου
Worksheet worksheet = workbook.Worksheets["Sheet1"];
```

## Βήμα 6: Πρόσβαση σε ένα συγκεκριμένο κελί

 Αφού πλοηγηθείτε στο επιθυμητό φύλλο εργασίας, μπορείτε να πλοηγηθείτε σε ένα συγκεκριμένο κελί χρησιμοποιώντας το`Cells` ιδιοκτησία του`Worksheet` αντικείμενο και ευρετηρίαση της αναφοράς κελιού.

```csharp
// Πρόσβαση σε ένα συγκεκριμένο κελί
Cell cell = worksheet.Cells["A1"];
```

## Βήμα 7: Ανάκτηση τιμής κελιού

 Τέλος, μπορείτε να ανακτήσετε την τιμή του κελιού χρησιμοποιώντας το`Value` ιδιοκτησία του`Cell` αντικείμενο.

```csharp
// Ανακτήστε την τιμή του κελιού

```
