Σε αυτό το σεμινάριο, θα σας δείξουμε πώς να καθορίσετε τον συγγραφέα κατά την προστασία εγγραφής ενός βιβλίου εργασίας του Excel χρησιμοποιώντας τη βιβλιοθήκη Aspose.Cells για .NET.

## Βήμα 1: Προετοιμασία του περιβάλλοντος

Πριν ξεκινήσετε, βεβαιωθείτε ότι έχετε εγκατεστημένο το Aspose.Cells για .NET στον υπολογιστή σας. Κατεβάστε τη βιβλιοθήκη από τον επίσημο ιστότοπο της Aspose και ακολουθήστε τις οδηγίες εγκατάστασης που παρέχονται.

## Βήμα 2: Διαμόρφωση καταλόγων προέλευσης και εξόδου

Στον παρεχόμενο πηγαίο κώδικα, πρέπει να καθορίσετε τους καταλόγους προέλευσης και εξόδου. Τροποποιήστε το`sourceDir` και`outputDir` μεταβλητές αντικαθιστώντας το "YOUR SOURCE DIRECTORY" και το "YOUR OUTPUT DECTORY" με τις αντίστοιχες απόλυτες διαδρομές στο μηχάνημά σας.

```csharp
// Κατάλογος πηγής
string sourceDir = "PATH TO YOUR SOURCE DIRECTORY";

// Κατάλογο εξόδου
string outputDir = "YOUR OUTPUT DIRECTORY PATH";
```

## Βήμα 3: Δημιουργία άδειου βιβλίου εργασίας του Excel

Για να ξεκινήσουμε, δημιουργούμε ένα αντικείμενο βιβλίου εργασίας που αντιπροσωπεύει ένα κενό βιβλίο εργασίας του Excel.

```csharp
// Δημιουργία κενού βιβλίου εργασίας.
Workbook wb = new Workbook();
```

## Βήμα 4: Γράψτε προστασία με κωδικό πρόσβασης

 Στη συνέχεια, καθορίζουμε έναν κωδικό πρόσβασης για να γράψουμε προστασία του βιβλίου εργασίας του Excel χρησιμοποιώντας το`WriteProtection.Password` ιδιοκτησία του αντικειμένου του βιβλίου εργασίας.

```csharp
// Γράψτε το βιβλίο εργασίας προστασίας με κωδικό πρόσβασης.
wb.Settings.WriteProtection.Password = "YOUR_PASSWORD";
```

## Βήμα 5: Προδιαγραφή συγγραφέα

 Τώρα καθορίζουμε τον συγγραφέα του βιβλίου εργασίας του Excel χρησιμοποιώντας το`WriteProtection.Author` ιδιοκτησία του αντικειμένου του βιβλίου εργασίας.

```csharp
// Καθορίστε τον συγγραφέα κατά τη σύνταξη του βιβλίου εργασίας προστασίας.
wb.Settings.WriteProtection.Author = "YOUR_AUTHOR";
```

## Βήμα 6: Δημιουργία αντιγράφων ασφαλείας προστατευμένου βιβλίου εργασίας του Excel

 Μόλις καθοριστεί η προστασία εγγραφής και ο συγγραφέας, μπορούμε να αποθηκεύσουμε το βιβλίο εργασίας του Excel σε μορφή XLSX χρησιμοποιώντας το`Save()` μέθοδος.

```csharp
// Αποθηκεύστε το βιβλίο εργασίας σε μορφή XLSX.
wb.Save(outputDir + "outputSpecifyAuthorWhileWriteProtectingWorkbook.xlsx");
```

### Δείγμα πηγαίου κώδικα για Specify Author while Write Protecting Excelbook Workbook using Aspose.Cells for .NET 
```csharp
//Κατάλογος πηγής
string sourceDir = "YOUR SOURCE DIRECTORY";

//Κατάλογο εξόδου
string outputDir = "YOUR OUTPUT DIRECTORY";

// Δημιουργία κενού βιβλίου εργασίας.
Workbook wb = new Workbook();

// Γράψτε το βιβλίο εργασίας προστασίας με κωδικό πρόσβασης.

// Καθορίστε τον συγγραφέα κατά τη σύνταξη του βιβλίου εργασίας προστασίας.
wb.Settings.WriteProtection.Author = "YOUR_AUTHOR";

// Αποθηκεύστε το βιβλίο εργασίας σε μορφή XLSX.
wb.Save(outputDir + "outputSpecifyAuthorWhileWriteProtectingWorkbook.xlsx");

```

#### Ε: Πώς μπορώ να καταργήσω την προστασία εγγραφής από ένα βιβλίο εργασίας του Excel;

#### Ε: Ξέχασα τον κωδικό πρόσβασης για την προστασία του βιβλίου εργασίας του Excel. Τι μπορώ να κάνω ?

Α: Εάν ξεχάσατε τον κωδικό πρόσβασης για την προστασία του βιβλίου εργασίας του Excel, δεν μπορείτε να τον καταργήσετε απευθείας. Ωστόσο, μπορείτε να δοκιμάσετε να χρησιμοποιήσετε εξειδικευμένα εργαλεία τρίτων που παρέχουν δυνατότητες ανάκτησης κωδικού πρόσβασης για προστατευμένα αρχεία Excel.

#### Ε: Είναι δυνατόν να καθοριστούν πολλοί συντάκτες κατά την προστασία εγγραφής ενός βιβλίου εργασίας του Excel;

Α: Όχι, η βιβλιοθήκη Aspose.Cells για .NET επιτρέπει τον καθορισμό ενός μόνο συγγραφέα κατά την προστασία εγγραφής ενός βιβλίου εργασίας του Excel. Εάν θέλετε να καθορίσετε πολλούς συντάκτες, θα πρέπει να εξετάσετε προσαρμοσμένες λύσεις χειρίζοντας απευθείας το αρχείο Excel.
