इस गाइड में, हम आपको बताएंगे कि .NET के लिए Aspose.Cells का उपयोग करके एक्सेल स्प्रेडशीट में प्रिंट शीर्षक कैसे सेट करें। इस कार्य को पूरा करने के लिए नीचे दिए गए चरणों का पालन करें।

## चरण 1: वातावरण स्थापित करना

सुनिश्चित करें कि आपने अपना विकास परिवेश स्थापित कर लिया है और .NET के लिए Aspose.Cells स्थापित कर लिया है। आप लाइब्रेरी का नवीनतम संस्करण Aspose की आधिकारिक वेबसाइट से डाउनलोड कर सकते हैं।

## चरण 2: आवश्यक नामस्थान आयात करें

अपने C# प्रोजेक्ट में, Aspose.Cells के साथ काम करने के लिए आवश्यक नामस्थान आयात करें:

```csharp
using Aspose.Cells;
```

## चरण 3: दस्तावेज़ निर्देशिका के लिए पथ सेट करना

 घोषित करें ए`dataDir` उस निर्देशिका का पथ निर्दिष्ट करने के लिए वैरिएबल जहां आप जेनरेट की गई एक्सेल फ़ाइल को सहेजना चाहते हैं:

```csharp
string dataDir = "YOUR_DIRECTORY_OF_DOCUMENTS";
```

 प्रतिस्थापित करना सुनिश्चित करें`"YOUR_DOCUMENT_DIRECTORY"` आपके सिस्टम पर सही पथ के साथ.

## चरण 4: एक कार्यपुस्तिका ऑब्जेक्ट बनाना

एक वर्कबुक ऑब्जेक्ट को इंस्टेंट करें जो उस एक्सेल वर्कबुक का प्रतिनिधित्व करता है जिसे आप बनाना चाहते हैं:

```csharp
Workbook workbook = new Workbook();
```

## चरण 5: पहली वर्कशीट तक पहुंच

निम्नलिखित कोड का उपयोग करके एक्सेल वर्कबुक में पहली वर्कशीट पर जाएँ:

```csharp
Worksheet worksheet = workbook.Worksheets[0];
```

## चरण 6: शीर्षक कॉलम को परिभाषित करना

निम्नलिखित कोड का उपयोग करके शीर्षक कॉलम परिभाषित करें:

```csharp
pageSetup.PrintTitleColumns = "$A:$B";
```

यहां हमने कॉलम ए और बी को शीर्षक कॉलम के रूप में परिभाषित किया है। आप इस मान को अपनी आवश्यकताओं के अनुसार समायोजित कर सकते हैं।

## चरण 7: शीर्षक पंक्तियों को परिभाषित करना

निम्नलिखित कोड का उपयोग करके शीर्षक पंक्तियों को परिभाषित करें:

```csharp
pageSetup.PrintTitleRows = "$1:$2";
```

हमने पंक्तियों 1 और 2 को शीर्षक पंक्तियों के रूप में परिभाषित किया है। आप इन मूल्यों को अपनी आवश्यकताओं के अनुसार समायोजित कर सकते हैं।

## चरण 8: एक्सेल कार्यपुस्तिका को सहेजना

 एक्सेल वर्कबुक को परिभाषित प्रिंट शीर्षकों के साथ सहेजने के लिए, इसका उपयोग करें`Save` कार्यपुस्तिका ऑब्जेक्ट की विधि:

```csharp
workbook.Save(dataDir + "SetPrintTitle_out.xls");
```

यह एक्सेल वर्कबुक को निर्दिष्ट निर्देशिका में फ़ाइल नाम "SetPrintTitle_out.xls" के साथ सहेजेगा।

### .NET के लिए Aspose.Cells का उपयोग करके सेट एक्सेल प्रिंट शीर्षक के लिए नमूना स्रोत कोड 
```csharp
//दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";
// वर्कबुक ऑब्जेक्ट को इंस्टेंट करना
Workbook workbook = new Workbook();
// वर्कशीट के पेजसेटअप का संदर्भ प्राप्त करना
Aspose.Cells.PageSetup pageSetup = workbook.Worksheets[0].PageSetup;
// कॉलम संख्या ए और बी को शीर्षक कॉलम के रूप में परिभाषित करना
pageSetup.PrintTitleColumns = "$A:$B";
// पंक्ति संख्या 1 और 2 को शीर्षक पंक्ति के रूप में परिभाषित करना
pageSetup.PrintTitleRows = "$1:$2";
// कार्यपुस्तिका सहेजें.
workbook.Save(dataDir + "SetPrintTitle_out.xls");
```

## निष्कर्ष

बधाई हो! आपने सीखा है कि .NET के लिए Aspose.Cells का उपयोग करके Excel स्प्रेडशीट में प्रिंट शीर्षक कैसे सेट करें। प्रिंट शीर्षक आपको प्रत्येक मुद्रित पृष्ठ पर विशिष्ट पंक्तियों और स्तंभों को प्रदर्शित करने की अनुमति देते हैं, जिससे डेटा को पढ़ना और संदर्भ देना आसान हो जाता है।

### पूछे जाने वाले प्रश्न

#### 1. क्या मैं एक्सेल में विशिष्ट कॉलम के लिए प्रिंट शीर्षक सेट कर सकता हूँ?

 हां, .NET के लिए Aspose.Cells के साथ आप विशिष्ट कॉलम को प्रिंट शीर्षक के रूप में सेट कर सकते हैं`PrintTitleColumns` की संपत्ति`PageSetup` वस्तु।

#### 2. क्या कॉलम और प्रिंट पंक्ति शीर्षक दोनों को परिभाषित करना संभव है?

 हां, आप इसका उपयोग करके प्रिंट कॉलम और पंक्ति शीर्षक दोनों सेट कर सकते हैं`PrintTitleColumns` और`PrintTitleRows` के गुण`PageSetup` वस्तु।

#### 3. मैं .NET के लिए Aspose.Cells के साथ अन्य कौन सी लेआउट सेटिंग्स को अनुकूलित कर सकता हूं?

.NET के लिए Aspose.Cells के साथ, आप विभिन्न पेज लेआउट सेटिंग्स को कस्टमाइज़ कर सकते हैं, जैसे मार्जिन, पेज ओरिएंटेशन, प्रिंट स्केल और बहुत कुछ।
