इस ट्यूटोरियल में, हम आपको चरण दर चरण दिखाने जा रहे हैं कि .NET के लिए Aspose.Cells का उपयोग करके एक्सेल में हेडर और फ़ुटर कैसे सेट करें। हम प्रक्रिया को स्पष्ट करने के लिए C# स्रोत कोड का उपयोग करेंगे।

## चरण 1: वातावरण स्थापित करना

सुनिश्चित करें कि आपकी मशीन पर .NET के लिए Aspose.Cells स्थापित है। अपने पसंदीदा विकास परिवेश में एक नया प्रोजेक्ट भी बनाएं।

## चरण 2: आवश्यक पुस्तकालय आयात करें

अपनी कोड फ़ाइल में, Aspose.Cells के साथ काम करने के लिए आवश्यक लाइब्रेरी आयात करें। यहाँ संबंधित कोड है:

```csharp
using Aspose.Cells;
```

## चरण 3: डेटा निर्देशिका सेट करें

वह डेटा निर्देशिका सेट करें जहां आप संशोधित एक्सेल फ़ाइल को सहेजना चाहते हैं। निम्नलिखित कोड का प्रयोग करें:

```csharp
string dataDir = "YOUR DATA DIRECTORY";
```

संपूर्ण निर्देशिका पथ निर्दिष्ट करना सुनिश्चित करें.

## चरण 4: कार्यपुस्तिका और कार्यपत्रक बनाना

एक नई वर्कबुक ऑब्जेक्ट बनाएं और निम्नलिखित कोड का उपयोग करके वर्कबुक में पहली वर्कशीट पर नेविगेट करें:

```csharp
Workbook excel = new Workbook();
PageSetup pageSetup = excel.Worksheets[0].PageSetup;
```

यह वर्कशीट के साथ एक खाली वर्कबुक बनाएगा और उस वर्कशीट के पेजसेटअप ऑब्जेक्ट तक पहुंच प्रदान करेगा।

## चरण 5: हेडर सेट करना

 का उपयोग करके स्प्रेडशीट हेडर सेट करें`SetHeader` पेजसेटअप ऑब्जेक्ट की विधियाँ। यहाँ एक नमूना कोड है:

```csharp
pageSetup.SetHeader(0, "&A");
pageSetup.SetHeader(1, "&\"Times New Roman,Bold\"&D-&T");
pageSetup.SetHeader(2, "&\"Times New Roman,Bold\"&12&F");
```

यह हेडर में क्रमशः वर्कशीट का नाम, वर्तमान दिनांक और समय और फ़ाइल नाम सेट करेगा।

## चरण 6: पाद लेख परिभाषित करना

 का उपयोग करके स्प्रेडशीट फ़ुटर सेट करें`SetFooter` पेजसेटअप ऑब्जेक्ट की विधियाँ। यहाँ एक नमूना कोड है:

```csharp
pageSetup.SetFooter(0, "Hello World! &\"Courier New\"&14 123");
pageSetup.SetFooter(1, "&P");
pageSetup.SetFooter(2, "&N");
```

यह क्रमशः एक टेक्स्ट स्ट्रिंग, वर्तमान पृष्ठ संख्या और पाद लेख में पृष्ठों की कुल संख्या निर्धारित करेगा।

## चरण 7: संशोधित कार्यपुस्तिका को सहेजना

निम्नलिखित कोड का उपयोग करके संशोधित कार्यपुस्तिका सहेजें:

```csharp
excel.Save(dataDir + "OutputFileName.xls");
```

यह संशोधित कार्यपुस्तिका को निर्दिष्ट डेटा निर्देशिका में सहेज लेगा।

### .NET के लिए Aspose.Cells का उपयोग करके एक्सेल हेडर और फ़ुटर सेट करने के लिए नमूना स्रोत कोड 
```csharp
//दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";
// वर्कबुक ऑब्जेक्ट को इंस्टेंट करना
Workbook excel = new Workbook();
// वर्कशीट के पेजसेटअप का संदर्भ प्राप्त करना
PageSetup pageSetup = excel.Worksheets[0].PageSetup;
// हेडर के बाएँ भाग में वर्कशीट का नाम सेट करना
pageSetup.SetHeader(0, "&A");
//हेडर के मध्य भाग में वर्तमान दिनांक और वर्तमान समय निर्धारित करना
// और हेडर का फ़ॉन्ट बदल रहा है
pageSetup.SetHeader(1, "&\"Times New Roman,Bold\"&D-&T");
// हेडर के दाएँ भाग पर वर्तमान फ़ाइल नाम सेट करना और बदलना
// हेडर का फ़ॉन्ट
pageSetup.SetHeader(2, "&\"Times New Roman,Bold\"&12&F");
// पाद लेख के बाएँ भाग पर एक स्ट्रिंग सेट करना और फ़ॉन्ट बदलना
// इस स्ट्रिंग के एक भाग का ("123")
pageSetup.SetFooter(0, "Hello World! &\"Courier New\"&14 123");
// पादलेख के मध्य भाग में वर्तमान पृष्ठ क्रमांक सेट करना
pageSetup.SetFooter(1, "&P");
// पादलेख के दाएँ भाग पर पृष्ठ संख्या निर्धारित करना
pageSetup.SetFooter(2, "&N");
// कार्यपुस्तिका सहेजें.
excel.Save(dataDir + "SetHeadersAndFooters_out.xls");
```


## निष्कर्ष

अब आपने सीख लिया है कि .NET के लिए Aspose.Cells का उपयोग करके एक्सेल में हेडर और फ़ुटर कैसे सेट करें। इस ट्यूटोरियल ने आपको पर्यावरण की स्थापना से लेकर संशोधित कार्यपुस्तिका को सहेजने तक प्रक्रिया के हर चरण के बारे में बताया। अपनी Excel फ़ाइलों में और अधिक हेरफेर करने के लिए Aspose.Cells की विशेषताओं को और अधिक जानने के लिए स्वतंत्र महसूस करें।

### अक्सर पूछे जाने वाले प्रश्न (FAQ)

#### 1. मैं अपने सिस्टम पर .NET के लिए Aspose.Cells कैसे स्थापित कर सकता हूं?
.NET के लिए Aspose.Cells को स्थापित करने के लिए, आपको Aspose की आधिकारिक वेबसाइट से इंस्टॉलेशन पैकेज डाउनलोड करना होगा और दस्तावेज़ में दिए गए निर्देशों का पालन करना होगा।

#### 2. क्या यह विधि एक्सेल के सभी संस्करणों के साथ काम करती है?
हां, .NET के लिए Aspose.Cells के साथ हेडर और फ़ुटर सेट करने की विधि एक्सेल के सभी समर्थित संस्करणों के साथ काम करती है।

#### 3. क्या मैं हेडर और फ़ुटर को और अधिक अनुकूलित कर सकता हूँ?
हां, Aspose.Cells हेडर और फ़ूटर को अनुकूलित करने के लिए टेक्स्ट प्लेसमेंट, रंग, फ़ॉन्ट, पेज नंबर और बहुत कुछ सहित सुविधाओं की एक विस्तृत श्रृंखला प्रदान करता है।

#### 4. मैं हेडर और फ़ूटर में गतिशील जानकारी कैसे जोड़ सकता हूँ?
आप हेडर और फ़ूटर में वर्तमान दिनांक, समय, फ़ाइल नाम, पृष्ठ संख्या इत्यादि जैसी गतिशील जानकारी जोड़ने के लिए विशेष चर और फ़ॉर्मेटिंग कोड का उपयोग कर सकते हैं।

#### 5. क्या मैं हेडर और फ़ुटर को सेट करने के बाद हटा सकता हूँ?
 हाँ, आप इसका उपयोग करके शीर्षलेख और पादलेख हटा सकते हैं`ClearHeaderFooter` की विधि`PageSetup` वस्तु। यह डिफ़ॉल्ट शीर्षलेख और पादलेख पुनर्स्थापित करेगा.
