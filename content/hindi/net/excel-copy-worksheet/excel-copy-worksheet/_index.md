इस गाइड में, हम बताएंगे कि .NET के लिए Aspose.Cells लाइब्रेरी का उपयोग करके एक्सेल वर्कशीट को कैसे कॉपी किया जाए। हम आपको C# स्रोत कोड प्रदान करेंगे और इस कार्य को पूरा करने के लिए आवश्यक चरणों के बारे में बताएंगे। अंत में, हम आपको अपेक्षित परिणाम दिखाएंगे। आरंभ करने के लिए नीचे दिए गए निर्देशों का पालन करें।

## चरण 1: तैयारी

शुरू करने से पहले, सुनिश्चित करें कि आपने .NET के लिए Aspose.Cells इंस्टॉल कर लिया है और अपने पसंदीदा एकीकृत विकास वातावरण (IDE) में एक C# प्रोजेक्ट बनाया है। यह भी सुनिश्चित करें कि आपके पास उस एक्सेल फ़ाइल की एक प्रति है जिसमें आप हेरफेर करना चाहते हैं।

## चरण 2: आवश्यक पुस्तकालय आयात करें

 अपनी C# स्रोत फ़ाइल में, Aspose.Cells का उपयोग करके आवश्यक लाइब्रेरी आयात करें`using` निर्देश:

```csharp
using Aspose.Cells;
```

## चरण 3: फ़ाइल पथ सेट करें

 घोषित करें ए`dataDir` वैरिएबल बनाएं और इसे अपनी Excel फ़ाइल वाली निर्देशिका से प्रारंभ करें। उदाहरण के लिए :

```csharp
string dataDir = "PATH_TO_YOUR_DOCUMENT_DIRECTORY";
```

 प्रतिस्थापित करना सुनिश्चित करें`"PATH_TO_YOUR_DOCUMENT_DIRECTORY"` आपकी निर्देशिका के वास्तविक पथ के साथ।

## चरण 4: मौजूदा एक्सेल फ़ाइल लोड करें

 उपयोग`Workbook` मौजूदा एक्सेल फ़ाइल को खोलने के लिए Aspose.Cells से क्लास। उपयोग`InputPath` फ़ाइल पथ निर्दिष्ट करने के लिए चर:

```csharp
string InputPath = dataDir + "book1.xls";
Workbook wb = new Workbook(InputPath);
```

 सुनिश्चित करें कि आपने प्रतिस्थापित कर दिया है`"book1.xls"` आपकी एक्सेल फ़ाइल के वास्तविक नाम के साथ।

## चरण 5: वर्कशीट की प्रतिलिपि बनाएँ

 अब हम मौजूदा वर्कशीट को एक नई वर्कशीट में कॉपी करेंगे। उपयोग`Worksheets` की संपत्ति`Workbook` कार्यपत्रकों के संग्रह तक पहुँचने के लिए ऑब्जेक्ट:

```csharp
WorksheetCollection sheets = wb.Worksheets;
```

 फिर उपयोग करें`AddCopy` निर्दिष्ट वर्कशीट की प्रतिलिपि बनाने की विधि। उदाहरण के लिए, "शीट1" की प्रतिलिपि बनाने के लिए:

```csharp
sheets.AddCopy("Sheet1");
```

## चरण 6: एक्सेल फ़ाइल सहेजें

 उपयोग`Save` की विधि`Workbook` किसी नई फ़ाइल में परिवर्तन सहेजने के लिए ऑब्जेक्ट:

```csharp
wb.Save(dataDir + "CopyWithinWorkbook_out.xls");
```

आउटपुट फ़ाइल के लिए वांछित पथ और फ़ाइल नाम निर्दिष्ट करना सुनिश्चित करें।

### .NET के लिए Aspose.Cells का उपयोग करके एक्सेल कॉपी वर्कशीट के लिए नमूना स्रोत कोड 

```csharp
//दस्तावेज़ निर्देशिका का पथ.
string dataDir = "YOUR DOCUMENT DIRECTORY";
string InputPath = dataDir + "book1.xls";
// कोई मौजूदा Excel फ़ाइल खोलें.
Workbook wb = new Workbook(InputPath);
// के संदर्भ में एक वर्कशीट ऑब्जेक्ट बनाएं
// कार्यपुस्तिका की शीट.
WorksheetCollection sheets = wb.Worksheets;
// किसी मौजूदा शीट से डेटा को एक नई शीट में कॉपी करें
// कार्यपुस्तिका के भीतर शीट.
sheets.AddCopy("Sheet1");
// एक्सेल फ़ाइल सहेजें.
wb.Save(dataDir + "CopyWithinWorkbook_out.xls");
```

## निष्कर्ष

बधाई हो! अब आपने सीख लिया है कि .NET के लिए Aspose.Cells का उपयोग करके एक्सेल वर्कशीट को कैसे कॉपी किया जाता है। इस चरण-दर-चरण मार्गदर्शिका में दिखाया गया है कि आवश्यक पुस्तकालयों को कैसे आयात करें, मौजूदा एक्सेल फ़ाइल को लोड करें, वर्कशीट की प्रतिलिपि बनाएँ और संशोधित फ़ाइल को सहेजें। एक्सेल फ़ाइलों में कुशलतापूर्वक हेरफेर करने के लिए अपनी परियोजनाओं में इस पद्धति का उपयोग करने के लिए स्वतंत्र महसूस करें।
