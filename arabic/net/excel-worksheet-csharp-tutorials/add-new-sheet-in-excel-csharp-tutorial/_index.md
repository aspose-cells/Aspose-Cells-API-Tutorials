في هذا البرنامج التعليمي ، سنشرح التعليمات البرمجية المصدر C # خطوة بخطوة لإضافة ورقة جديدة في Excel باستخدام Aspose.Cells for .NET. تعد إضافة ورقة عمل جديدة إلى مصنف Excel عملية شائعة عند إنشاء التقارير أو معالجة البيانات. Aspose.Cells هي مكتبة قوية تجعل من السهل التعامل مع ملفات Excel وإنشاءها باستخدام .NET. اتبع الخطوات أدناه لفهم وتنفيذ هذا الرمز.

## الخطوة 1: إعداد دليل المستند

تتمثل الخطوة الأولى في تحديد دليل المستند حيث سيتم حفظ ملف Excel. إذا كان الدليل غير موجود ، نقوم بإنشائه باستخدام الكود التالي:

```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENTS DIRECTORY";
// قم بإنشاء الدليل إذا لم يكن موجودًا بالفعل.
bool IsExists = System.IO.Directory.Exists(dataDir);
if (!IsExists)
System.IO.Directory.CreateDirectory(dataDir);
```

تأكد من استبدال "دليل المستندات" بالمسار المناسب إلى دليل المستندات.

## الخطوة 2: إنشاء كائن مصنف

الخطوة الثانية هي إنشاء مثيل لكائن مصنف ، والذي يمثل مصنف Excel. استخدم الكود التالي:

```csharp
Workbook workbook = new Workbook();
```

سيتم استخدام هذا الكائن لإضافة ورقة عمل جديدة وتنفيذ عمليات أخرى في مصنف Excel.

## الخطوة 3: إضافة ورقة عمل جديدة

الخطوة الثالثة هي إضافة ورقة عمل جديدة إلى كائن المصنف. استخدم الكود التالي:

```csharp
int index = workbook. Worksheets. Add();
Worksheet worksheet = workbook.Worksheets[index];
```

سيؤدي هذا إلى إضافة ورقة عمل جديدة إلى كائن المصنف وستحصل على مرجع إلى ورقة العمل هذه باستخدام الفهرس الخاص بها.

## الخطوة 4: تعيين اسم ورقة العمل الجديدة

الخطوة الرابعة هي تسمية ورقة العمل الجديدة. يمكنك استخدام الكود التالي لتعيين اسم ورقة العمل:

```csharp
worksheet.Name = "My Worksheet";
```

استبدل "جدول البيانات" بالاسم المطلوب للورقة الجديدة.

## الخطوة 5: حفظ ملف Excel

أخيرًا ، الخطوة الأخيرة هي حفظ ملف Excel. استخدم الكود التالي:

```csharp
string filePath = dataDir + "output.out.xls";
workbook.Save(filePath);
```

سيؤدي هذا إلى حفظ مصنف Excel مع ورقة العمل الجديدة في دليل المستندات الذي حددته.

### نموذج التعليمات البرمجية المصدر لإضافة ورقة جديدة في برنامج Excel C # التعليمي باستخدام Aspose.Cells for .NET 
```csharp
// المسار إلى دليل المستندات.
string dataDir = "YOUR DOCUMENT DIRECTORY";
// قم بإنشاء دليل إذا لم يكن موجودًا بالفعل.
bool IsExists = System.IO.Directory.Exists(dataDir);
if (!IsExists)
	System.IO.Directory.CreateDirectory(dataDir);
// إنشاء كائن مصنف
Workbook workbook = new Workbook();
// إضافة ورقة عمل جديدة إلى كائن المصنف
int i = workbook.Worksheets.Add();
// الحصول على مرجع ورقة العمل المضافة حديثًا عن طريق تمرير فهرس الورقة الخاص بها
Worksheet worksheet = workbook.Worksheets[i];
// تعيين اسم ورقة العمل المضافة حديثًا
worksheet.Name = "My Worksheet";
// حفظ ملف Excel
workbook.Save(dataDir + "output.out.xls");
```

نعم ، يمكنك تطبيق التنسيق على الخلايا باستخدام الطرق التي توفرها فئة ورقة العمل الخاصة بـ Aspose.Cells. يمكنك ضبط نمط الخلية وتغيير لون الخلفية وتطبيق الحدود وما إلى ذلك.

#### كيف يمكنني الوصول إلى بيانات الخلية من ورقة العمل الجديدة؟

يمكنك الوصول إلى بيانات الخلية باستخدام الخصائص والطرق التي توفرها فئة ورقة العمل الخاصة بـ Aspose.Cells. على سبيل المثال ، يمكنك استخدام الخاصية Cells للوصول إلى خلية معينة واسترداد قيمتها أو تعديلها.
